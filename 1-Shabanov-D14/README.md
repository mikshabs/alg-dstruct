D14 - 'Судоку'
Решите головоломку судоку размером N > 2.
В текстовом файле input.txt храниться натуральное число 2 < N < 10. После чего записаны N*N строк, содержащие N*N целых чисел в интервале [0; N*N] каждая. (0 - означает, что клетка пуста, иначе там записано правильное значение). Выведите любое решение судоку в текстовый файл output.txt в том же формате, что и исходный файл. Если решения не существует выведите 0.
